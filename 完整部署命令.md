# 完整部署命令

## 📍 本地 Windows 电脑

### 1. 查看 IP
```bash
ipconfig
```
找到 IPv4 地址，例如：`10.100.36.40`

### 2. 编辑配置文件
打开 `cert-config.json`，确认 IP 在列表中：
```json
"ip": [
  "127.0.0.1",
  "10.100.36.40"
]
```

### 3. 安装依赖
```bash
npm install
```

### 4. 生成证书
```bash
npm run cert
```

### 5. 启动服务
```bash
npm start
```

### 6. 访问
- 本地：`https://localhost:3001`
- Vision Pro：`https://10.100.36.40:3001`

## ☁️ 云服务器（Ubuntu/Linux）

### 1. 连接服务器
```bash
ssh root@123.45.67.89
```

### 2. 查看服务器 IP
```bash
curl ifconfig.me
```
或
```bash
ip addr show
```

### 3. 安装 Node.js
```bash
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install -y nodejs
node --version
```

### 4. 安装 PM2
```bash
sudo npm install -g pm2
```

### 5. 配置防火墙
```bash
sudo apt install ufw -y
sudo ufw allow 22
sudo ufw allow 3001
sudo ufw enable
sudo ufw status
```

### 6. 在本地编辑配置文件
打开 `cert-config.json`，添加服务器 IP：
```json
"ip": [
  "127.0.0.1",
  "10.100.36.40",
  "123.45.67.89"
]
```

### 7. 在本地生成证书
```bash
npm run cert:force
```

### 8. 上传代码到服务器
```bash
scp -r ./* root@123.45.67.89:/root/3d-model-viewer/
```

### 9. 在服务器上安装依赖
```bash
ssh root@123.45.67.89
cd /root/3d-model-viewer
npm install
```

### 10. 启动服务
```bash
pm2 start server.js --name "3d-model-viewer"
pm2 startup
pm2 save
```

### 11. 查看状态
```bash
pm2 status
pm2 logs 3d-model-viewer
```

### 12. 访问
`https://123.45.67.89:3001`

## 🌐 云服务器 + 域名（可选）

### 1. 域名解析
在域名管理后台添加 A 记录：
```
类型: A
主机记录: @
记录值: 123.45.67.89
```

### 2. 安装 Nginx 和 Certbot
```bash
sudo apt install nginx certbot python3-certbot-nginx -y
```

### 3. 获取 SSL 证书
```bash
sudo certbot certonly --standalone -d 3dmodel.com
```

### 4. 配置 Nginx
```bash
sudo nano /etc/nginx/sites-available/3dmodel
```

粘贴以下内容：
```nginx
server {
    listen 80;
    server_name 3dmodel.com;
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name 3dmodel.com;

    ssl_certificate /etc/letsencrypt/live/3dmodel.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/3dmodel.com/privkey.pem;

    location / {
        proxy_pass https://localhost:3001;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

### 5. 启用配置
```bash
sudo ln -s /etc/nginx/sites-available/3dmodel /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
sudo systemctl enable nginx
```

### 6. 开放端口
```bash
sudo ufw allow 80
sudo ufw allow 443
```

### 7. 设置证书自动续期
```bash
sudo crontab -e
```
添加：
```
0 2 * * * certbot renew --quiet --post-hook "systemctl reload nginx"
```

### 8. 访问
`https://3dmodel.com`

## 🔧 常用管理命令

### PM2
```bash
pm2 list                           # 查看所有进程
pm2 logs 3d-model-viewer          # 查看日志
pm2 restart 3d-model-viewer       # 重启
pm2 stop 3d-model-viewer          # 停止
pm2 delete 3d-model-viewer        # 删除
```

### Nginx
```bash
sudo nginx -t                      # 测试配置
sudo systemctl restart nginx       # 重启
sudo systemctl status nginx        # 查看状态
```

### 防火墙
```bash
sudo ufw status                    # 查看状态
sudo ufw allow 端口号              # 开放端口
sudo ufw delete allow 端口号       # 关闭端口
```

### 证书
```bash
npm run cert                       # 生成证书
npm run cert:force                 # 强制重新生成
npm run cert:check                 # 检查证书状态
```

## 🐛 故障排查

### 无法访问
```bash
# 检查服务是否运行
pm2 status

# 检查端口是否监听
sudo netstat -tlnp | grep 3001

# 检查防火墙
sudo ufw status

# 查看日志
pm2 logs 3d-model-viewer
```

### 证书错误
```bash
# 重新生成证书
npm run cert:force

# 检查证书内容
openssl x509 -in cert/cert.pem -text -noout
```

### 服务崩溃
```bash
# 查看错误日志
pm2 logs 3d-model-viewer --err

# 重启服务
pm2 restart 3d-model-viewer
```
